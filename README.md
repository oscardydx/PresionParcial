# PresionParcial
Trabajo academico computacional con el fin de estudiar la entropía.

# Ejecución

Programa normal: 
  make
  
Testing:
  make test

Reporte gprof:
 make gprof

Reporte cachegrind:
  make cachegrind

Valgrind:
  make memcheck

Tiempo en función del tamaño del sistema:
  make tiempos

Limpiar
  make clean
